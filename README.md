# News Media Portal

A simple **News Media Portal** built using **HTML, CSS, and
JavaScript**. The project demonstrates a basic news website structure
with multiple pages, article views, and an admin page.

------------------------------------------------------------------------

## Features

-   Homepage displaying the main news sections
-   Individual article pages
-   Single article detailed view
-   Admin page layout
-   Responsive design
-   Organized assets, CSS, and JavaScript files

------------------------------------------------------------------------

## Project Structure

News-Media-Portal │ ├── Assets/ \# Images and media used in the project
├── css/ \# Stylesheets ├── js/ \# JavaScript files ├── index.html \#
Homepage ├── About.html \# About page ├── Article.html \# Page listing
multiple articles ├── SingleArticle.html \# Individual article view ├──
Admin.html \# Admin interface layout └── README.md \# Project
documentation

------------------------------------------------------------------------

## Getting Started

1.  Clone the repository

git clone https://github.com/stha-prasun/News-Media-Portal.git

2.  Open the project folder.

3.  Run the project by opening **index.html** in your browser.
